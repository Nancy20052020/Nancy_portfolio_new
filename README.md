# TREADURE — Nancy Verma Portfolio

Treasure-hunt style personal portfolio for Nancy Verma.

## Stack

- Next.js (App Router)
- Tailwind CSS
- GSAP + Framer Motion
- Light / dark parchment themes

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
