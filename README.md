# TREADURE — Nancy Verma Portfolio

Treasure-hunt style personal portfolio for Nancy Verma. Built with Next.js for Vercel.

## Stack

- Next.js 16 (App Router)
- Tailwind CSS
- GSAP + Framer Motion
- Light / dark parchment themes

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Import this GitHub repo in the [Vercel dashboard](https://vercel.com/new)
2. Framework Preset: **Next.js** (auto-detected)
3. Build Command: `npm run build`
4. Output: default Next.js output (no extra config needed)
5. Click **Deploy**

Or with the Vercel CLI:

```bash
npx vercel
```

Node.js 20+ is required (`engines` in `package.json`).
